##### React Native Hjri Calendar (Picker)



###### Usage
```
import RnHjriDatePicker from "rn-hjri-calendar";
<RnHjriDatePicker
        onDone={d => {
          setShowModal(false);
          setDate(d);
        }}
        visible={showModal}
        doneText={"تم"}
        sep1style={{}}
        sep2style={{}}
        closeText={"اغلاق"}
        yearsTitle={"اختر السنة"}
      />
```
